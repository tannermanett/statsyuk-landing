The Database

Our system integrates multiple data sources, including NHL APIs, Moneypuck, and PuckPedia, capturing play-by-play events, salary cap details, player performance metrics, and historical game data. The backend is built in Python, leveraging SQLAlchemy for database management, Pandas and NumPy for data processing, and Requests, Selenium and BeautifulSoup for API integration and web scraping. The database is powered by PostgreSQL, set up on NeonDB, ensuring high scalability and performance for handling large volumes of hockey data. The frontend is being developed with React, using Chart.js to create interactive and visually compelling data visualizations that allow users to explore detailed player stats, game events, and performance trends.

But were not done yet! Our long-term goal is to build AI and deep learning models using the rich dataset we are collecting. This includes predictive analytics for player performance, team strategy optimization, and advanced machine learning techniques to model game outcomes, player development, and contract efficiency. By combining traditional statistics with AI, we aim to create the most advanced yet accessible NHL analytics tool, catering to teams, analysts, and hockey enthusiasts alike.
